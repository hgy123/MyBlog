### 1.

```
Happiness is good.
幸福就是至善。
```

### 2.

```
Beauty is a natural advantage.
美是一种自然优势。
```

### 3.

```
Knowledge is human nature.
求知是人类的本性。
```

### 4.

```
Happiness comes from own.
幸福来源于我们自己。
```

### 5.

```
I love my teacher, I love the true.
吾爱吾师，吾尤爱真理。
```

### 6.

```
Habits can make second nature.
习惯早就第二天性。
```
